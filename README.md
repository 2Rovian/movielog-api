# MovieLog API

REST API built with **Spring Boot** and **JPA/Hibernate** to manage movies.

## Features
- Create movies
- List movies
- Update movies
- Delete Movies
- Search movies by title and imdb rating
- Get movie by id
- Global exception handling

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA (Hibernate)
- H2 Database (in-memory)
- Lombok

Query params:
- `title` (optional)
- `imdb` (optional)
